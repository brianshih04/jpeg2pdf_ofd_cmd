## 🚀 使用步骤

### **方式 1：直接运行 JAR（最稳定）**

1. **打开命令提示符**
   - 按 `Win + R`
   - 输入 `cmd`
   - 按回车

2. **进入项目目录**
   ```bash
   cd D:\Projects\JPEG2PDF_OFD_Java_2
   ```

3. **运行服务**
   ```bash
   java -Xmx2G -jar target\jpeg2pdf-ofd-1.0.0.jar
   ```

4. **等待服务启动**
   - 看到 "Started Application" 表示成功
   - 等待约 20 秒

5. **打开浏览器**
   - 访问：http://localhost:8000
   - 或者直接运行：
     ```bash
     start http://localhost:8000
     ```

---

### **方式 2：使用便携版（推荐新手）**

1. **打开文件夹**
   ```
   进入：D:\Projects\JPEG2PDF_OFD_Java_2\dist\JPEG2PDF-OFD-Portable
   ```

2. **双击运行**
   ```
   双击：启动服务.bat
   ```

3. **等待启动**
   - 会弹出命令提示符窗口
   - 等待约 20 秒
   - 看到 "服务启动成功" 消息息

4. **浏览器自动打开**
   - 如果没有自动打开，手动访问：http://localhost:8000

---

### **方式 3：使用 jpackage 版（无需安装 Java）**

1. **打开文件夹**
   ```
   进入：D:\Projects\JPEG2PDF_OFD_Java_2\dist\JPEG2PDF-OFD
   ```

2. **双击运行**
   ```
   双击：JPEG2PDF-OFD.exe
   ```

3. **等待启动**
   - 等待约 20 秒
   - 命令提示符窗口会出现

4. **打开浏览器**
   - 讌问：http://localhost:8000
